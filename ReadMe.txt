
На всякий случай, игра запускается через файл "Annoying Haru.exe"


Если у вас Windows 11, то возможно наличие консольного окна (подробности ниже). В таком случае, просто сверните его, но не закрывайте. 
Проект написан для Windows 10, поэтому гарантировать стабильность работы на других ОС не могу.


"BackroundMusicApp.exe" - это мини-приложение, которое управляется игрой и служит для воспроизведения
			  фоновой музыки. НЕ ПЫТАЙТЕСЬ ЗАПУСКАТЬ ЕГО! так как завершить процесс можно
			  будет только через диспетчер задач. По этой же причине, не рекомендуется
			  закрывать игру некорректным способом.
